Sobre ScanXTCG:

ScanXTCG es una página web donde podrás consultar el valor de tus cartas de TCG según Cardmarket. Funciona con Pokémon, MTG, Lorcana y muchas más.

Desarrollado originalmente por Markel Cantón usando solo Visual Studio Code, mediante el nombre o código de la carta, podrás consultar en tiempo real el valor de tus cartas de TCG, teniendo de la misma manera tu inventario actualizado.

¿Qué hace a ScanXTCG único?

  • Estilo visual único: usamos imágenes para mantener el contenido visual activo.
  • Actualizaciones periódicas: trabajamos mucho para mantener ScanXTCG activo y bien desarrollado. Con el tiempo, iremos implementando nuevas funciones.
  • Interfaz limpia y original: ScanXTCG usa la misma interfaz que PoKuiz, otro proyecto de Markel Cantón. Ambas han sido desarrolladas desde cero usando solo Visual Studio Code.
