# ScanXTCG

ScanXTCG es una aplicación web desarrollada por Markel Cantón para calcular el valor de cartas de Juegos de Cartas Coleccionables (o TCGs) de acuerdo a las estadísticas de <a target="_blank" title="https://cardmarket.com" href="https://cardmarket.com">Cardmarket</a>.
